# AI Keyboard

基于 Figma 设计生成的智能键盘组件，支持 Web 和移动端响应式布局。

## 功能特性

- 精美的暗色主题设计
- 流畅的动画效果
- 响应式布局
- 支持多种输入模式（小写/大写/符号）
- 键盘事件监听

## 快速预览

```bash
npx serve .
```
